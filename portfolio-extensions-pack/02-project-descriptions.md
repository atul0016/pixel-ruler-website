# Project Descriptions

## 1) Advanced Smart Capture

### One-liner
A complete screenshot workflow extension for Chrome with full-page capture, OCR, annotation tools, and export options in one place.

### Short Description
Advanced Smart Capture helps users capture full pages, selected regions, or specific elements directly in Chrome. It includes annotation tools, OCR text extraction, and export options like PNG/JPG/PDF, making it a practical all-in-one productivity tool for developers, creators, and teams.

### Portfolio Description (Long)
Advanced Smart Capture is a feature-rich Chrome extension I built to unify the entire screenshot workflow inside the browser. Instead of using multiple tools for capture, annotation, and text extraction, users can perform everything in one extension.

The extension supports full-page capture, visible area capture, region selection, and element-level capture. It also includes built-in annotation capabilities such as arrows, text, highlight, blur, and shape tools. OCR support allows users to extract text from screenshots, while export options include PNG, JPG, and PDF.

I designed this project with a product mindset: clear free/premium feature boundaries, practical UX for repeated usage, and a lightweight Chrome MV3 architecture.

### Key Highlights
- Full-page and region capture modes
- OCR text extraction from screenshots
- Annotation toolkit (text, arrows, shapes, blur)
- Export formats: PNG, JPG, PDF
- Scheduled capture support
- Chrome Web Store published listing

## 2) Pixel Ruler + Alignment Checker

### One-liner
A precision layout extension that overlays rulers, guides, and alignment tools on webpages for developers and designers.

### Short Description
Pixel Ruler + Alignment Checker is a Chrome extension focused on UI precision. It provides on-screen measurement aids such as rulers, guides, and alignment checks to help developers/designers verify spacing and layout consistency directly on live webpages.

### Portfolio Description (Long)
Pixel Ruler + Alignment Checker is a practical developer tool I built to speed up visual QA and front-end alignment work. The extension overlays measurement and alignment assistance directly on webpages, helping users validate spacing, element positioning, and design consistency without leaving the browser.

This tool is especially useful during responsive UI checks, handoff validation, and fast front-end debugging sessions. The goal was to keep it lightweight, intuitive, and instantly useful for daily workflow.

### Key Highlights
- On-page ruler and measurement support
- Alignment guide functionality
- Useful for UI QA and design consistency checks
- Lightweight browser-based workflow
- Chrome Web Store published listing
