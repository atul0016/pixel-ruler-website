# Chrome Extensions Portfolio Pack

This folder contains ready-to-use content for your portfolio website for both of your Chrome extensions.

## Extensions Covered

1. Advanced Smart Capture
2. Pixel Ruler + Alignment Checker

## Official Chrome Web Store Links

- Advanced Smart Capture: https://chromewebstore.google.com/detail/advanced-smart-capture/iepjkkadegefnikifdmppccceobbbeli
- Pixel Ruler + Alignment Checker: https://chromewebstore.google.com/detail/pixel-ruler-+-alignment-c/mlkckmpmigjabkmodhflghieipignmie

- Short and long project descriptions
- Hero copy for homepage
- Feature highlights
- Suggested screenshots and captions
- Portfolio section structure
- SEO metadata
- CTA copy
