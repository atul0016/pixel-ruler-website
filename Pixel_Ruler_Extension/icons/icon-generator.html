📋 How to Generate Icons

Follow these steps to create the required PNG icons for your Chrome extension:

  1. Online SVG to PNG Converter:
    • Visit Convertio or CloudConvert
    • Upload the icon.svg file from this folder
    • Generate PNG files at these exact sizes: 16x16, 48x48, and 128x128 pixels
  2. Using Figma/Adobe Illustrator:
    • Import the SVG file
    • Export as PNG at the required sizes
    • Make sure to maintain the aspect ratio
  3. Using Command Line (ImageMagick):
# Install ImageMagick first # Convert SVG to required PNG sizes magick icon.svg -resize 16x16 icon16.png magick icon.svg -resize 48x48 icon48.png magick icon.svg -resize 128x128 icon128.png

    📁 Required Files

    Your icons/ folder should contain these files when complete:

    icons/ ├── icon16.png (16x16 pixels) ├── icon48.png (48x48 pixels) ├── icon128.png (128x128 pixels) └── icon.svg (source file)

    🎯 Icon Design Guidelines

    • Style: Clean, modern, recognizable as a ruler tool
    • Colors: Primary blue (#007acc), accent orange (#ff6b35)
    • Elements: Rulers, grid lines, measurement indicators
    • Background: Solid color or subtle gradient
    • Visibility: Must be clear at 16px size
